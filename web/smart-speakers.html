Android quick-actions + Google Routine

"Hey Google, book Servia"

Honest scope: Google doesn't expose a one-tap routine import API the way Apple does for Siri. There's no .routine file we can host. The closest two paths that DO work today on Android:

⚡ Path 1 — PWA install ➜ long-press the icon (one tap setup)

Install Servia from the home page (or Install page). On Android, long-press the Servia app icon on your home screen — these 4 quick-actions appear instantly:

Already declared in manifest.webmanifest. Works on Chrome / Edge / Samsung Browser on Android. No keyboard shortcut creation needed.

Setup steps (1 minute)
  1. Open Google Home app (or Assistant settings on Android)
  2. Routines → + Add a routine
  3. Trigger: "When I say 'Hey Google, book Servia'"
  4. Action: choose Open app or website → paste https://servia.ae/search.html
