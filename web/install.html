📋 How to install on a Wear OS watch

Wear OS watches don't accept APK files directly from a download — they need ADB sideload (one-time setup) until we publish on Play Store. Three minutes total:

  1. On the watch: Settings → About → tap "Build number" 7× to unlock Developer Options. Then Settings → Developer Options → enable ADB debugging + Debug over Wi-Fi. Note the IP shown.
  2. On a computer: install Android Platform Tools (free, ~10 MB).
  3. Download the watch APK to your computer (the orange button above).
  4. Connect: adb connect <watch-IP>:5555
  5. Install: adb -s <watch-IP>:5555 install servia-wear-v*.apk
  6. Servia appears on the watch's app list. Open → tap any tile → opens the full Servia experience (auto-mirrors via the phone if the watch is offline).

🚀 Coming soon: Once we publish to the Wear OS Play Store ($25 one-time fee, ~3 day review), watch users can install with one tap from Play Store on the watch — no ADB needed.

❌ NOT compatible with:

Apple Watch (different OS — separate WatchOS app coming, see iOS card below) · Garmin / Fitbit / Amazfit / Huawei Watch GT (proprietary OS, no Android app support) · Wear OS 1 / Wear OS 2 watches older than 2018 (Galaxy Watch 1, Apple-style classic Wear devices — APK won't install).
