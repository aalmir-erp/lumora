How it works (advance-payment policy)

  1. Customer books → bot calls create_booking() → status pending_payment.
  2. System auto-issues invoice + Payment Link via the configured provider.
  3. Customer pays via card / Apple Pay / Google Pay / WhatsApp link.
  4. Provider webhook hits /api/webhooks/stripe → invoice marked paid → booking auto-confirmed → crew dispatched.
  5. If unpaid in 30 minutes, automatic reminder; in 2 hours, slot auto-released.

🗣 Voice integrations — Google Home / Alexa / Siri

Customers' custom SOS shortcuts can be controlled by voice via Google Home (Smart Home Action), Apple Siri Shortcuts, and IFTTT routines. Servia hosts the OAuth + fulfillment endpoints; setup happens once in Google's Actions Console and then customers self-link in the Google Home app.

📋 One-time setup in Actions on Google

  1. Go to console.actions.google.com → New project → name it Servia SOS.
  2. Pick Smart Home as the project type. Skip the optional fields.
  3. Develop → Invocation: set display name to Servia (this is what users say: "Hey Google, ask Servia…"). Pick a male voice.
  4. Develop → Account linking:
    • Account creation: No, only allow account creation on my website
    • Linking type: OAuth · Grant type: Authorization code
    • Client ID: paste from above
    • Client secret: paste from above
    • Authorization URL: https://servia.ae/oauth/authorize
    • Token URL: https://servia.ae/oauth/token
    • Scopes: leave blank (or enter shortcuts)
  5. Develop → Actions → Add a new Action → Smart Home:
    • Fulfillment URL: https://servia.ae/api/google-home/fulfillment
  6. Develop → Account linking → Testing instructions: enter Use any UAE phone number to register at https://servia.ae/sos.html, save 1 SOS shortcut, then link.
  7. Test tab → Account linking testing → click Test. Google opens our consent UI; approve. The Action should now appear in your Google Home app under "Works with Google → Servia".
  8. Open Google Home app → + → Set up device → Works with Google → search Servia → link → all your custom SOS shortcuts appear as scenes.
  9. Test voice: "Hey Google, run [your shortcut name]". The shortcut fires; truck dispatched.
  10. When happy, hit Deploy → Submit for review in the console. 3-5 day Google review for public listing.
